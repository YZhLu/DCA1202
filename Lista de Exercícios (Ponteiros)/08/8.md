8_ O que fazem os seguintes programas em C?
```
int main(){
    int vet[] = {4,9,13};
    int i;
    for(i=0;i<3;i++){
        printf("%d ",*(vet+i));
    }
}

int main(){
    int vet[] = {4,9,13}; 
    int i; 
    for(i=0;i<3;i++){
        printf("%X ",vet+i);
    }
}
```

Resposta:

'''
int main(){
    int vet[] = {4,9,13};
    int i;
    for(i=0;i<3;i++){
        printf("%d ",*(vet+i));
    }
}
'''
